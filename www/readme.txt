Перенос данных с  PostgreSql в Mysql

Структура исходной  БД должна  быть  обновлена  до  версии 6.9.0. (обновления в  папке  pgupdate )


1. Создать Mysql базу (кодировка utf8_general_ci)
2. Выполнить db_nodata.sql 
3. Положить в  корень  сайта слада  файл pg2mysql.php
4. Прописать в  начале  файл соединения с  исходной и целевой БД
5. Открыть в  браузере http://<имя сайта>>/pg2mysql.php
6. Сменить конект в  config.php  и перелогинится

Ориентировочно  правильность данных можно  оценииь  по  совпажению цифр на  инфоблоках вверху дашборда
